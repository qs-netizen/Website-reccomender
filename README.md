# 🌐 Website Recommender (Flask)

A simple Python web app that finds **similar websites** based on a user search.
The app uses **Flask**, **Requests**, and **BeautifulSoup** to search the web and recommend related sites.

---

## 🚀 Features

* 🔎 Search any website or topic
* 🌍 Finds similar websites automatically
* 🧠 Filters out blog/article pages
* 🎨 Clean UI built with HTML + CSS
* 🐍 Entire project written in **Python**

---

## 🛠️ Tech Stack

* Python
* Flask
* Requests
* BeautifulSoup
* HTML / CSS

---

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/website-recommender.git
cd website-recommender
```

2. Install dependencies

```bash
pip install flask requests beautifulsoup4
```

---

## ▶️ Running the App

Start the Flask server:

```bash
python ml.py
```

Then open your browser and go to:

```
http://127.0.0.1:5000
```

---

## 🔍 Example Searches

Try searching for:

* youtube
* netflix
* wikipedia
* amazon

The app will return **similar websites**.
